Montagu Slab Variable Font
==========================

This download contains Montagu Slab as both a variable font and static fonts.

Montagu Slab is a variable font with these axes:
  opsz
  wght

This means all the styles are contained in a single file:
  Montagu_Slab/MontaguSlab-VariableFont_opsz,wght.ttf

Get started
-----------

1. Install the font files you want to use

2. Use your app's font picker to view the font family and all the
available styles
