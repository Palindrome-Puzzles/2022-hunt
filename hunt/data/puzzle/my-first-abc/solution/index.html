Each clue leads to a phrase that represents a letter from a children's alphabet book. (Author's initials are given below the puzzle in alphabetical order by title.) Each book provides two letters (e.g., Sandra Boynton's A to Z gives both O and W.) Starting with the given letters and swapping them for the other letter from the same book yields SI THOUSAND PREFIX which defines the answer: KILO.
