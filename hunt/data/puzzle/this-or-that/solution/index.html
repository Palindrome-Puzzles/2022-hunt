The theme of this puzzle is alternates, as clued by the title. The answer to each clue has an alternate answer with the same meaning that is used in British English instead. This is also hinted by the flavortext phrase "where you're coming from". Each pair of answers fit a given pair of cell enumerations, which are in alphabetical order of the American English answers on the left. The indexed letters from each pair form the last name of the alter ego of a DC Comics character. The corresponding DC character name has the same length (and word breaks) as the words in its clue, which serves as a confirmation. Furthermore, each clue uses an instance of the word "this" or "that", and extracting the corresponding letters from those positions in the DC character names, in the given order, gives the clue phrase "ALTERNATE IN MARVEL".
