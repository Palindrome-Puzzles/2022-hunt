Each puzzle answer clues a famous person. The eleven names can be entered into the grid, crisscross-style. The direction of the arrow indicates the direction the name goes in that row or column, and the side of the grid the arrow and number are on is the side the name starts closer to. The fact that the grid is 12 rows by 31 columns (and that some rows have black boxes to shorten them to either 28 or 30 boxes) should indicate that it represents a calendar. Each person's name crosses the square on the grid that represents that person's birthday. Taking the letters in those squares and ordering using the numbers by the arrows spells A WEB OF LIVES.
