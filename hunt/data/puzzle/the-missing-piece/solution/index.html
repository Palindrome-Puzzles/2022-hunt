These are typical examples of Palindrome name badges; therefore each team name is depicted as a palindrome pictorially. (The name badges are presented in order by palindrome length.)

The years on the name badges give the alpha ordering for the team names.

The count of the symbols repeated at the top of each name badge marks the enumeration for the palindromic team names.

Each tag also has one or two of those symbols partially punched by a hole; any symbol touched by a hole is an answer letter to extract.

And the number of times each name badge lanyard overlaps itself determines the ordering for the extracted answer letters:
